# test の勉強をするためのプロジェクト

## 作るもの

-   数字を 2 つ入力するとグラデーションアニメーションする
-   数字は２つ保持されて、古い数字から新しい数字にグラデーションする
-   3 つ目以降の入力は古いものが押し出される形で保持される
-   10 進数で入力されると 16 進数に変換されて保持される
-   16 進数で入力する場合は「#」から始まるカラーコードを入力する
-   古いカラーコードとカラーを表示
-   新しいカラーコードとカラーを表示
-   css コードを表示する
